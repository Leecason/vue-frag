Vue 2 fragment directive to unwrap root elements

## 🙋‍♂️ Why?
- **🔥 Multiple root nodes** Without creating a functional component!
- **⭐️ SSR** Unwraps the root element on client-side post-hydration!
- **🐥 Tiny** Only `277 B`!

## 🚀 Install
```sh
npm i vue-frag
```
